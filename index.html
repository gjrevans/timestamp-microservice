Timestamp Microservice

Ever wanted to be able to convert a UNIX time a natural time or vice versa, who hasn't!?

You're in luck, this handy timestamp microservice accepts a either a natural time or UNIX time and will return a JSON object of both values. If an unrecognized date format is provided the API will return null for both values in the object.

Example Useage

https://timestamp-service-api.herokuapp.com/December%2015,%202015
https://timestamp-service-api.herokuapp.com/1450137600

Example output:

{ "unix": 12345678, "natural": "January 12, 3456" }
